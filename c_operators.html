Typecasting in C

Typecasting is a way to convert a variable from one data type to another. In C, it is necessary when you want to perform operations between different data types, such as between integers and floating-point numbers, or when you want to force a conversion to a specific type.

C supports two types of typecasting:

  • Implicit Typecasting: This occurs automatically when data is converted from a smaller data type to a larger one. For example, an integer can be automatically converted to a float in an arithmetic operation.
  • Explicit Typecasting: This is manually done by the programmer using a cast operator to convert one data type to another. This typecasting is necessary when narrowing conversions (from larger to smaller types) are required or when mixing types in an expression.

The syntax for explicit typecasting is: (type) expression, where type is the desired data type, and expression is the variable or value to be cast.

Example:

int a = 5;
    float b = 2.3;
    int result = (int)b + a;  // Explicit cast from float to int

Booleans in C

The C language does not have a dedicated boolean data type in older versions (before C99). However, starting from C99, C introduced the `_Bool` data type, which can store the values 0 (false) and 1 (true).

A more commonly used approach before C99 was to treat integers as booleans, where 0 represented false, and any non-zero value represented true.

In C99 and later, you can include the <stdbool.h> header to use the more readable `bool`, `true`, and `false` keywords.

Example:

#include <stdbool.h>
    
    bool is_valid = true;
    if (is_valid) {
        // Do something if true
    }

Without <stdbool.h>, you can use integers:

int is_valid = 1;  // 1 for true, 0 for false
    if (is_valid) {
        // Do something if true
    }
