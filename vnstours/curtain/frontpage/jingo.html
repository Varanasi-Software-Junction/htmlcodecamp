Walkthrough: Building a Responsive Front Page for an E-commerce Shop Using CSS Grid

Step 1: Basic Structure

We start by creating the basic HTML structure with a header, main content area, and a footer. The main content contains sections for categories and featured products.

<header>
<h1>Shop Name</h1>
<nav>
<ul>
<li><a href="#">Home</a></li>
<li><a href="#">Products</a></li>
<!-- Other navigation items -->
</ul>
</nav>
</header>
<!-- Main content with categories and products -->
<footer><p>© 2024 Shop Name. All rights reserved.</p></footer>

Illustration 1: Basic HTML structure

Step 2: CSS Grid Layout

Next, we define the grid layout for categories and products using CSS Grid. The number of columns adjusts based on the screen size.

.grid-container {
    display: grid;
    gap: 20px;
}

@media (min-width: 768px) {
    .categories .grid-container {
        grid-template-columns: repeat(3, 1fr);
    }
    .products .grid-container {
        grid-template-columns: repeat(4, 1fr);
    }
}

Illustration 2: Grid layout for categories and products

Step 3: Responsive Layout with Media Queries

We add media queries to make the layout responsive. On smaller screens, categories and products will display in a single-column format.

@media (max-width: 767px) {
    .categories .grid-container,
    .products .grid-container {
        grid-template-columns: 1fr;
    }
}

Illustration 3: Responsive layout using media queries

Step 4: Enhancements for Better UX

To make the user experience more interactive, we add hover effects to products and categories. This simple CSS enhancement improves visual appeal.

.product-item:hover,
.category-item:hover {
    transform: scale(1.05);
    transition: transform 0.3s ease;
}

Illustration 4: Adding hover effects for products and categories

Step 5: Product Information and Images

Include images and product information (name, price, etc.) to enhance the design. This step makes the product section more functional and visually engaging.

<div class="product-item">
    <img src="path-to-product.jpg" alt="Product Image">
    <h3>Product Name</h3>
    <p>$29.99</p>
</div>

Illustration 5: Adding images and product details

Step 6: Footer Design

Finally, we complete the design with a simple footer containing copyright information. You can also add social media links and contact information here.

<footer>
    <p>© 2024 Shop Name. All rights reserved.</p>
</footer>

Illustration 6: Footer design
